# hello-world-autotools-template

This is a very simple C project template using [autotools build system](https://www.gnu.org/software/automake/manual/html_node/Autotools-Introduction.html).


## Setup

Initialize scripts

`autoreconf --install`

Configure

`./configure`

Make & install

`make && make install`
